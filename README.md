# DDE_Dock_Weather

This is a fork from [sonichy](https://github.com/sonichy/WEATHER_DDE_DOCK) with theme options feature. 

Deepin Linux DDE-Dock weather forcast plugin.  
[Releases](../../releases/)  
install：./install.sh  
uninstall：./uninstall.sh  

or manually compile it: 
```
mkdir build
cd build
qmake ..
make 
sudo make install
```
